Docker

Docker는 Container에 특정 OS와 해당 os에 구축된 환경에 대한 image를 적재하고 해당 Container를 통해 여러 명령어 및 프로그램을 수행하게 해줌. 결국, 다양한 OS및 환경 구축에 대한 부분을 프로그래머가 직접 하기 보다 다른 이들이 구축해둔 image를 활용함으로서 보다 편리한 개발이 가능해질 것으로 보임.
