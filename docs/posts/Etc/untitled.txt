국어 :
* 큰 고민 없음

수학 : 
* 미분 II, 기하와 벡터. 시 이과학생.
* 고급수학의 경우, 높은 수준이나 일반고의 경우 제대로 진행되려면 수강자 수가 적어야 함.
  * 고급수학 시간에 무엇을 배웠는지 확인해야 함.

영어 :
* 큰 고민 없음

사회 :
* 문과 학생의 경우, 과목의 선택권이 없음.
* 심화 과정은 국제고 과정, 단 지나치게 이름이 너무 거창하면 선수과목이 있는지를 살펴야 함. 조합으로 기본과정과 심화과정이 있는지 확인해야함.
* 과제 연구 과목이 많아지고 있음.
  * 13명 이상시 등급을 매겨야 하므로, 해당 13명 이하로 쪼개고 있음.
  
과학 :
* 과학 1은 이과학생이면 다함.
* 과학 2은 과중반 아니면 그리 많지 않음. : 실질적 선택이 있음.
  * 일부 학교에서는 과학2 시간에 과학1 을 복습하는 경우가 있음. : 세특 에서 확인해야 함.
  * 물리2에서 torque를 배우지 않음.
* 고급 과 실험 은 과학고 수준. : 지나치게 이수자 수가 많으면 수준을 확인해야 함.(과중반 학생이 아닌 경우, 이름만...)


어떤 과목이든 2단위로만 채운 경우, 제대로 배운 경우가 아니기 쉬움.

교양은 평가 자체가 없음. 앉아만 있으면 이수.
